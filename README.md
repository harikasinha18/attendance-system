# 🎓 Advanced Student Attendance Eligibility System

An interactive and responsive Student Attendance Eligibility System developed using **HTML, CSS, and JavaScript**.

This project allows users to generate student attendance records dynamically, check eligibility based on attendance percentage, search students, download reports, and use dark mode for better UI experience.

---

## 🚀 Features

✅ Dynamic Student Table Generation  
✅ Attendance Eligibility Checking  
✅ Responsive Modern UI  
✅ Dark Mode Toggle  
✅ Student Search Functionality  
✅ Attendance Progress Bars  
✅ Download Attendance Report  
✅ Local Storage Support  
✅ Show / Hide Table Option  
✅ Stylish Animated Interface  

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
- Git & GitHub
